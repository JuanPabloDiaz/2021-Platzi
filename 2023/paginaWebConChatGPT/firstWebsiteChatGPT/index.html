Curso de Creación de Páginas Web con ChatGPT

¿Necesitas crear una página web, pero no cuentas con experiencia y desconoces las herramientas? Desarrolla y publica una Página Web sin conocimientos en programación utilizando ChatGPT y comparte tu portafolio profesional. Adquiere experiencia en el uso de ChatGPT, junto a Estefany Aguilar, Senior Frontend Developer en La Haus.

  • Realiza un proyecto web utilizando ChatGPT como herramienta de apoyo.
